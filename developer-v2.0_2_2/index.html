Major League Baseball Attendance Analysis

What factors contribute to baseball attendance? Can we accurately predict attendance for Los Angeles Dodgers games?

New

Major League Baseball, America's "favorite pastime", faced an inconvenient truth throughout the 2018 season -- fans weren’t showing up. League wide attendance is down 6.5% and the lowest it has been in 15 years. This is a major concern for all stakeholders with commercial interest[s] tied to strong attendance numbers. The goal of this project was to analyze why baseball attendance is on the decline, and what relevant factors affect attendance the most. This analysis helped us determine what strategies MLB's stakeholders might consider to increase attendance and revenue.

Utilized MySQL, SQLAlchemy, and Pandas library in Python to process/manipulate data; created a random forest regressor model to calculate feature importance—then used NumPy and Matplotlib libraries as well as Tableau to create data visualizations and display findings.

Language/Technologies/Tools:
  • Data storage: MySQL
  • Data processing: Python (SQLAlchemy, PyMySQL, Pandas, NumPy)
  • Machine Learning: Scitkit-learn library (for Random Forest Regressor & Linear Regression models)
  • Data Visualizations/Dashboards: Matplotlib, Tableau, HTML, Javascript (Leaflet, Plotly, D3)
